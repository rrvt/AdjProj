Test App Operations

Open File (Toolbar Folder Image)

Opens a project file and display it in the window. Use scroll bar to see the whole file.

Fix Project File

Fix the loaded Project File and display resulting fixes. Scroll down to near the end of the file to see changes.

Save File As (Toolbar Floppy Image)

Displays a Save File Dialog and saves the file to the file named. Note that the old file is renamed with a trailing time stamp.

Print (Toolbar Printer Image)

Will do all the normal things to print a file.

About (Toolbar Question Mark)

Displays the About Box. The about box uses text from the resource file (TestApp.rc) for the output information. There is a standard about box dialog in the Library.

Exit (Toolbar X image)

Exit the application with no reservations.
